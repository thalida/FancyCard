### Development
##### Build + Run
```
npm start
gulp
```
Browse to the app at `http://localhost:8000/app/index.html`.


##### Deploy to Prod
```
git checkout deploy
git merge master
git push
git push prod deploy:master
```
Check that launch was successful at: `thalida.com`
